## 🌞 Деплой на GitHub:

### 1. Создайте репозиторий на GitHub:
1. Зайдите на https://github.com/new
2. Название: `contentgen-pro`
3. Описание: `AI Content Generator for Social Media and Marketplaces`
4. Публичный репозиторий
5. НЕ добавляйте README, .gitignore, лицензию (уже есть)

### 2. Свяжите локальный репозиторий:
```bash
# Замените на ваши данные
git remote add origin https://github.com/ваш-ник/contentgen-pro.git
git branch -M main
git push -u origin main
```

### 3. Готово! 
Ваш проект на GitHub: `https://github.com/ваш-ник/contentgen-pro`

## 🚀 Деплой на Vercel (бесплатный хостинг):

### Автоматический деплой:
1. Зайдите на https://vercel.com
2. Импортируйте GitHub репозиторий
3. Vercel автоматически определит настройки
4. Добавьте переменные окружения:
   - `GROQ_API_KEY`: ваш Groq API ключ
   - `JWT_SECRET`: любой секретный ключ

### Ручной деплой:
```bash
npm install -g vercel
vercel
```

## 🌟 Деплой на GitHub Pages (статическая версия):

### Для фронтенда только:
1. В настройках GitHub репозитория
2. Settings → Pages
3. Source: Deploy from branch
4. Branch: main
5. Folder: /public
6. Save

**Результат:** `https://ваш-ник.github.io/contentgen-pro`
