# ContentGen Pro - AI Генератор контента

🚀 **Готовый SaaS сервис для монетизации с настоящим AI**

## ✨ Что это?

**ContentGen Pro** - веб-сервис, который генерирует уникальный контент для бизнеса с помощью ChatGPT и Groq AI:
- Посты для соцсетей (Instagram, VK, Telegram)
- Описания товаров для маркетплейсов (Wildberries, Ozon)
- Разные тона общения (дружелюбный, профессиональный, с юмором, официальный)

## 💰 Модели монетизации

### Подписки (основной доход)
- **Бесплатный**: 50 генераций/месяц
- **Про**: 990₽/мес - безлимитные генерации
- **Бизнес**: 2990₽/мес - API, белый брендинг

### Быстрый старт заработка
1. **Фриланс-биржи** - настройка сервиса за 3000-5000₽
2. **SMM-специалисты** - подписки на контент-генерацию
3. **Контекстная реклама** - "генератор постов для соцсетей"

## 🛠 Технологии

### Бэкенд
- **Node.js + Express** - сервер
- **SQLite** - база данных пользователей
- **JWT** - аутентификация
- **Groq AI** - бесплатный Llama 3 API
- **Stripe** - платежи (опционально)

### Фронтенд
- **HTML/CSS/JavaScript** - без фреймворков
- **Tailwind CSS** - стилизация
- **Font Awesome** - иконки

## � Быстрый запуск

### 1. Локально (5 минут)
```bash
# Клонировать
git clone https://github.com/ваш-ник/contentgen-pro.git
cd contentgen-pro

# Установить зависимости
npm install

# Добавить Groq API ключ
cp .env.example .env
# GROQ_API_KEY=gsk_ваш_ключ

# Запустить
npm start
# http://localhost:3001
```

### 2. Vercel (бесплатный хостинг)
```bash
# Установить Vercel CLI
npm install -g vercel

# Развернуть
vercel

# Добавить GROQ_API_KEY в настройках Vercel
```

### 3. GitHub Pages (статическая версия)
```bash
# Залить на GitHub
git push origin main

# Включить Pages в настройках репозитория
```
